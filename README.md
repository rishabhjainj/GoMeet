
## Go Meet
Meeting Scheduling API STACK
### Technology Stack

> MongoDB  
> GoLang 

### Setup Instructions

1. Clone the repository
2. Install following packages  
      ```go get gopkg.in/mgo.v2 ```
3. Setup MongoDB
     Server : localhost
     Database : meetings_db
     Collection : meetings
     
     Port currently set to default mongoDB port : 27017
